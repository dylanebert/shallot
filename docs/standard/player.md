# Player

First-person character controller. Pointer lock, WASD movement, mouse look, sprint. When paired with `Character` (physics body), handles camera creation, readback interpolation, jump buffering, coyote time, and eye-height offset automatically. Without `Character`, works as a free-fly camera.
